Network-Based View of Conflicts

In this section, wars are modeled as a network: Countries are nodes and conflicts are links. two network visualization purposed : Bubble hierarchy and Sankey Diagram.

Section 5 — Key Network Insights

Sankey Diagram (Region → Region Flows)

A Sankey diagram showing aggregate flows between locations (regions) and war purposes. Link thickness encodes the number of conflicts for each region–purpose combination.

Sankey diagram

Bubble Hierarchy: War Purpose → Region

A circle-packing hierarchy where each large bubble represents a war purpose and the inner bubbles represent regions. Bubble size is proportional to the number of wars in each region–purpose subtree.
